## Tools Used
- VScode /  Assembly Language 

## Project Overview
- What does it implement?
It implements a simple encryp/decrypt tool using XOR technique

- How does it work?

This project uses the XOR encryption technique.
How XOR Works:
If two bits are same → result is 0
If two bits are different → result is 1
Formula:
Encrypted Text = Plain Text XOR Key
Decrypted Text = Encrypted Text XOR Key
The same operation is used for both encryption
and decryption
