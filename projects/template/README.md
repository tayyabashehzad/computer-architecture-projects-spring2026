## How to Submit Your Project

### Step 1 тАФ Fork this repository

Click the **Fork** button on the top right of this page.
This creates your own copy of the repo.

### Step 2 тАФ Clone your fork

```bash
git clone https://github.com/YOUR-USERNAME/computer-architecture-projects-spring2026.git
cd computer-architecture-projects-spring2026
```

### Step 3 тАФ Create your branch

```bash
git checkout -b feature/your-name-rollno
```

Example:
```bash
git checkout -b feature/ali-hassan-41600
```

### Step 4 тАФ Create your project folder

```bash
mkdir projects/ali-hassan-41600
```

> **Important:** Folder name must be `your-name-rollno` тАФ no spaces, use hyphens.

### Step 5 тАФ Add your project files

```
projects/ali-hassan-41600/
тФЬтФАтФА README.md           тЖР Required
тФЬтФАтФА code/               тЖР Your implementation files
тФЬтФАтФА report.pdf          тЖР Optional
тФФтФАтФА presentation.pptx  тЖР Optional
```

### Step 6 тАФ Commit and push

```bash
git add .
git commit -m "feat: add project by ali-hassan-41600"
git push origin feature/ali-hassan-41600
```

### Step 7 тАФ Create a Pull Request

1. Go to your forked repo on GitHub
2. Click **Compare & pull request**
3. Set **base repository** тЖТ `tayyabashehzad/computer-architecture-projects-spring2026`
4. Set **base branch** тЖТ `main`
5. Title: `Project submission - Ali Hassan - 41600`
6. Click **Create pull request**

## Common Mistakes to Avoid

| Wrong | Right |
|---|---|
| Pushing directly to main | Create a branch first |
| Empty folder | Add all project files |
| Missing README | Include README.md |
| Wrong folder name | Use `name-rollno` format |
| PR to wrong branch | Always target `main` |
